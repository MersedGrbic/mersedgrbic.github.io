Latest Project

Taskmate

Taskmate is a user-friendly to-do app that allows you to manage your tasks in an efficient manner. It uses Auth0 for secure login and registration, so you can access your account with ease. With Taskmate, you can create, edit, and delete tasks, ensuring that you stay organized and on top of your to-do list. Built using ReactJS, ExpressJS, NodeJS, and MongoDb, Taskmate is a fast and reliable tool that you can use on the go.

Taskmate is not fully responsive yet.
